# Task Manager API

Esta é uma API para gerenciar tarefas, permitindo que os usuários criem, leiam, atualizem e excluam tarefas. 

## Tecnologias Utilizadas

- **Node.js**: Ambiente de execução JavaScript do lado do servidor.
- **Express**: Framework para construir APIs de forma simples e rápida.
- **MongoDB**: Banco de dados NoSQL para armazenar tarefas.
- **Jest**: Framework de testes para JavaScript.

## Propósito

O objetivo deste projeto é facilitar a gestão de tarefas para usuários, oferecendo uma interface simples e intuitiva para interações com as tarefas.
